# Job: Decentralized Storage Researcher | Library Innovation Lab, Harvard University | September 2022

**Source:** [lil.law.harvard.edu](https://lil.law.harvard.edu/jobs/) · **Published:** 2022-09-12 · **Feed tags:** opportunities

> "The Harvard Library Innovation Lab seeks paid, contract researchers with substantial technical skills to help us explore and document cutting-edge data storage platforms. At the core of this research is the question: how can important data be stored for the long term with the least cost and risk? Who should apply? This is a great project for any technologist who enjoys exploring open source repos and testing how they perform in real world conditions; it may be particularly interesting to academic technologists or open source hackers looking for an excuse to learn more about peer-to-peer data storage and other strategies for long-term cultural preservation. What is the project? The project will include: Selecting a range of data storage technologies for analysis, including API storage providers like Amazon S3, established data archival platforms like LOCKSS, emerging data replication protocols like IPFS and hypercore, and emerging data storage offerings like Sia, Storj, and Filecoin...."

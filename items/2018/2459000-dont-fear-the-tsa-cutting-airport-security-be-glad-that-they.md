# Don’t fear the TSA cutting airport security. Be glad that they’re talking about it. - The Washington Post

**Source:** [washingtonpost.com](https://www.washingtonpost.com/news/posteverything/wp/2018/08/07/dont-fear-the-tsa-cutting-airport-security-be-glad-that-theyre-talking-about-it/?noredirect=on&utm_term=.4e0a1d3cda95) · **Published:** 2018-08-07 · **Feed tags:** website, buzz

> Don’t fear the TSA cutting airport security. Be glad that they’re talking about it.
> 
> Berkman Klein Fellow [Bruce Schneier](http://cyber.harvard.edu/people/bschneier)argues in [this piece](https://www.washingtonpost.com/news/posteverything/wp/2018/08/07/dont-fear-the-tsa-cutting-airport-security-be-glad-that-theyre-talking-about-it/?noredirect=on&utm_term=.25c303843fa8) for the Washington Post that despite largely negative media coverage of the subject, it is a good thing that TSA is brainstorming ways to optimize security in airports. “We need to evaluate airport security based on concrete costs and benefits, and not continue to implement security theater based on fear,” Schneier says.

# How About Machine Learning Enhancing Theses?

**Source:** [hamlet.andromedayelton.com](https://hamlet.andromedayelton.com/) · **Published:** 2018-09-27 · **Feed tags:** website, buzz

> How about machine learning enhancing theses?
> 
> Berkman Klein software developer [Andromeda Yelton](https://cyber.harvard.edu/people/andromeda-yelton) created “How about machine learning enhancing theses?” or [Hamlet](https://hamlet.andromedayelton.com/), a prototype tool for identifying literature and theses with similar conceptual topics. Hamlet uses machine learning to power experimental, exploratory interfaces to the MIT thesis collection.
